# Football Player Performance Analysis 

##  Project Overview
This project analyzes football player performance using data analytics techniques. The dataset includes player statistics such as goals, assists, and matches played.

##  Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

##  Key Analysis
- Top goal scorers
- Goals distribution
- Goals vs assists
- League comparison
- Correlation heatmap

##  Insights
- Players with more matches tend to score more goals
- Strong relationship between shots on target and goals
- Performance varies across leagues

##  Files Included
- Dataset (.xlsx)
- Jupyter Notebook (.ipynb)
- PPT

##  Conclusion
This project demonstrates how data analytics can be used to analyze sports performance and generate meaningful insights.
